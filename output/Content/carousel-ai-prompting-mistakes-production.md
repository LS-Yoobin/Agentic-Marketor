## Canva Build Order

1. Create `#0D1117` base template, 1080x1080
2. Build the Before/After column template once — reuse for slides 2–6
   - Two equal columns with thin gold vertical divider
   - Prompt box style: `#1A2332` background, rounded corners (8px), monospace font
3. Build Slide 9 (CTA) — base for non-mistake slides
4. Build Slides 2–5 (Before/After mistakes) — swap copy per slide
5. Build Slide 6 (GitHub mistake) — same template
6. Build Slide 7 (Core Truth) — single stat/insight slide
7. Build Slide 8 (The 3 Types) — framework reveal
8. Build Slide 1 (Hook) last
9. Export all as PNG, 300dpi

**Canva tip:** Build one Before/After slide perfectly, then duplicate it 4 times and only swap the copy. Saves 80% of the build time.
