# Social Media Content Strategy Workflow

> This workflow governs all social media content creation for Bloggo and the founder's personal brand.
> The agent executes this end-to-end. The founder reviews and approves output batches only.

---

## Accounts

| Account | Purpose |
|---------|---------|
| **Personal Brand** | Credibility, authority, and audience growth through founder journey + product builder content |
| **Bloggo Company Page** | Organic reach, brand identity, values, and product awareness |

---

## Platforms

| Platform | Accounts | Primary Format | Output Required |
|----------|----------|----------------|-----------------|
| Instagram | Both | Reels + Carousels | Script + visual direction + caption |
| TikTok | Both | Short-form viral videos | Same script as Instagram Reel |
| YouTube | Both | Shorts (repurposed) | Same script as Instagram Reel + title (max 100 chars) + description (max 5000 chars) + 3–5 hashtags |
| LinkedIn | Both | Founder posts + brand insight posts | Full post (hook + body + CTA) |

> **Efficiency rule:** Instagram Reels → TikTok → YouTube Shorts is one video, three platforms. Always produce short-form video once and route to all three. YouTube Shorts require a title, description, and hashtags in addition to the Reel script — always include these.

## Posting Cadence

### Bloggo Company Page
| Platform | Frequency | Format Mix |
|----------|-----------|------------|
| Instagram | 4x/week | 2 Reels + 2 Carousels |
| TikTok | 4x/week | Repurposed from Instagram Reels |
| YouTube | 4x/week | Shorts (same videos + YouTube-specific title/description/tags) |
| LinkedIn | 2x/week | 1 brand values post + 1 travel/product post |

### Personal Brand Page
| Platform | Frequency | Format Mix |
|----------|-----------|------------|
| Instagram | 3x/week | 2 Reels + 1 Carousel |
| TikTok | 3x/week | Repurposed from Instagram Reels |
| YouTube | 3x/week | Shorts (repurposed + YouTube-specific title/description/tags) |
| LinkedIn | 2x/week | Founder journey + product builder insight |

> **Default batch size:** Each content run produces one week of content — 6 unique pieces for Bloggo (2 Reel scripts + 2 Carousels + 2 LinkedIn posts) and 5 unique pieces for Personal Brand (2 Reel scripts + 1 Carousel + 2 LinkedIn posts). TikTok and YouTube Shorts receive the same Reel scripts — no additional drafting required for those platforms. The founder may request a smaller batch (e.g., "run 3 pieces only") and the agent will comply.

## Workflow Steps

Run these steps in order for every content batch.

### Step 0 — Read Active Launch Phase
Read the `Current phase` line above. Apply the matching CTA from the Launch Phase Modifier table to every piece of content in this run. Do not proceed without confirming the active phase.

---

### Step 1 — Check the Idea Inbox
Read `docs/Marketing/Idea Inbox.md`.

- If the inbox contains ideas: prioritize them above all AI-generated ideas. Match each to the closest pillar and best platform/format.
- If the inbox is empty: proceed directly to Step 2 and generate all ideas from research and pillar rotation.
- If any idea conflicts with brand rules: do not silently drop it. Flag it in a **Brand Rule Conflicts** section at the end of the batch output with a brief explanation and a suggested compliant alternative.

---

### Step 2 — Research Pulse
Run a web search on the following before generating ideas:
- Trending travel topics or viral travel moments this week
- AI app news relevant to on-device AI, privacy, or photo organization
- Competitor moves (Google Photos, Apple Memories, Day One, Polarsteps)
- Trending Reel/TikTok formats or audio in travel and startup niches
- Founder/startup discourse worth tapping into for the personal brand

Summarize findings in 3–5 bullets. These will be included in the **Research Findings** section of the batch output. Use findings to make content timely and culturally relevant. Cite anything specific.

---

### Step 3 — Load All Context

**Read `docs/Marketing/bloggo-brand.md` first, before any other file.** This is the non-negotiable foundation for all copy decisions.

Then read every remaining `.md` file in `docs/Marketing/`:

| File | What It Provides |
|------|-----------------|
| `bloggo-product-context.md` | Full product identity, features, personas, differentiators |
| `marketing-research.md` | Market data, competitive landscape |
| `Content Grader.md` | Hook scoring, grading rubric, brand checks |
| `Viral Short-Form Video Playbook.md` | Reel/TikTok/Short formula + Bloggo examples |
| `Instagram Carousel Cheat Sheet.md` | Carousel structure, hook types, slide-by-slide rules |

> No file is optional. All must be active before any content is written.

---

### Step 4 — Generate Content Ideas
Produce ideas to fill the default batch (or founder-specified size). Mix pillars across both accounts. Prioritize founder inbox ideas first.

For each idea, define:
- Account (Personal Brand / Bloggo)
- Pillar
- Platform(s)
- Format (Reel script / Carousel / LinkedIn post)
- Hook (first line only — validate the hook before writing the full piece)

---

### Step 5 — Write Drafts
Write full content for each idea, platform-native:

| Format | Account Voice | Output |
|--------|---------------|--------|
| **Reel / TikTok / Shorts** | Bloggo: brand voice. Personal Brand: founder's first-person voice. | Hook, visual direction, voiceover or text overlays, payoff, CTA |
| **YouTube Shorts extras** | Same as Reel | + Title (max 100 chars) + Description (max 5000 chars) + 3–5 hashtags |
| **Carousel** | Bloggo: brand voice. Personal Brand: founder's first-person voice. | Slide-by-slide copy, Slide 1 hook through Slide 10 CTA |
| **LinkedIn — Personal Brand** | Founder's first-person voice. Conversational, narrative, peer-to-peer. Not Bloggo brand voice. | Hook + story or insight body + CTA |
| **LinkedIn — Bloggo Page** | Bloggo brand voice from `bloggo-brand.md`. Benefit-first, warm, confident, simple. | Hook + body + CTA |

Apply the active launch phase CTA (from Step 0) to every piece.

---

### Step 6 — Grade Every Piece
Run each draft through the Content Grader rubric (`docs/Marketing/Content Grader.md`). Apply all three scoring tiers:

| Score | Verdict | Action |
|-------|---------|--------|
| 28–35 | **PUSH** | Include as-is |
| 20–27 | **REVISE** | Rewrite hook or restructure. Re-grade. Include revised version only. |
| Below 20 | **REWRITE** | Scrap and rewrite from scratch. Re-grade. Include rewritten version only. |

Flag any brand voice violations in a separate note beneath the piece's grade.

---

## Founder Review

After receiving the batch, the founder does one of three things:
1. **Approve** — content goes to scheduling
2. **Request changes** — specify what to adjust, agent revises and re-grades
3. **Inject a new idea** — drop it in the Idea Inbox or communicate directly; agent incorporates into the next batch

To change the launch phase, update the `Current phase` line in the **Launch Phase Modifier** section above before triggering the next run.

---

## Brand Rules (Always Active)

These apply to every piece of content, every platform, every time:

1. Never mention cloud features — Bloggo V1 is on-device only
2. iOS only — never imply Android availability
3. Privacy is a feature — lead with it when relevant
4. "Bloggo" is always capitalized
5. Active voice always — "Bloggo creates" not "blogs are created"
6. Lead with benefit, not feature
7. No jargon — "leverage," "utilize," "seamless," "powerful" are banned
8. Short sentences. One idea per sentence.
9. No more than one exclamation point per post
