## Agentic Flow Ideas

### 5 Automation Opportunities for Yoobin

---

**1. Area: Content Distribution**
**What to automate:** Auto-post finished carousel HTML screenshots to Instagram + TikTok on a schedule. Agent reads today's `output/Carousels/[DATE]/` folder, screenshots each slide at 360×450px using Playwright, batches them into a post, and schedules via Buffer or Later API at optimal posting time (7am, 12pm, or 7pm by engagement data).
**Tool/MCP:** Playwright MCP + Buffer API + filesystem MCP
**Time saved:** 45–60 min/day (manual screenshot + upload workflow)
**Difficulty:** Medium

---

**2. Area: Product — App Store Reputation**
**What to automate:** Monitor App Store reviews daily. Agent scrapes new reviews via App Store Connect API, runs sentiment + topic analysis (bug reports vs praise vs feature requests), and auto-creates a GitHub Issue or Notion card for anything mentioning a bug or a repeated pain point. Summarizes weekly in a Slack message.
**Tool/MCP:** App Store Connect API + GitHub MCP + Notion MCP
**Time saved:** 2–3 hrs/week (manually checking reviews + triaging)
**Difficulty:** Medium

---

**3. Area: Finance — Revenue Tracking**
**What to automate:** Daily pull of App Store Connect sales data → append to a Google Sheet with date, downloads, revenue, and conversion rate. Agent runs every morning and flags any day where revenue drops >20% vs 7-day average, sends a Slack alert. Replaces manual dashboard checking.
**Tool/MCP:** App Store Connect API + Google Sheets MCP + Slack MCP
**Time saved:** 30 min/day (manual revenue checking)
**Difficulty:** Easy

---

**4. Area: Research — Competitor Intelligence**
**What to automate:** Weekly scan of App Store pages for 5 key competitors (Day One, Polarsteps, Unfold, Canva, Journey). Agent checks: new screenshots, description changes, rating changes, new reviews mentioning pain points. Summarizes changes in a Notion page tagged `competitor-watch`. Triggers only when a change is detected.
**Tool/MCP:** WebFetch MCP + Notion MCP + diff comparison logic
**Time saved:** 1–2 hrs/week (manual competitor monitoring)
**Difficulty:** Easy

**5. Area: Health/Habits — Morning Standup Agent**

**Tool/MCP:** Notion MCP + GitHub MCP + Slack/iMessage MCP + filesystem MCP
**Time saved:** 20–30 min/day (scattered morning tab-opening ritual)
