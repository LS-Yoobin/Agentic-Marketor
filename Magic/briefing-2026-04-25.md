## Agentic Flow Ideas

> Based on Agentic-Marketor repo structure and Yoobin's profile as a solo co-founder building Bloggo.

### 1. Carousel Screenshot Automation Agent
- **Area:** Content Creation / Distribution
- **What to automate:** After carousel HTML files are generated each morning, a Puppeteer script auto-screenshots each slide at 360×450px, saves PNGs to `output/Carousels/[DATE]/slides/`, and queues them for Buffer or Later via API. Zero manual screenshotting.
- **Tool / MCP:** Puppeteer (Node) + Buffer API or Later API + Claude for caption generation
- **Time saved:** ~3 hours/week (screenshotting + uploading is currently the most manual step)
- **Difficulty:** Medium — Puppeteer setup is straightforward; Buffer API key needed

### 2. Daily Idea Inbox Voice Note Pipeline
- **Area:** Personal Productivity / Content
- **What to automate:** Yoobin records a 30-second voice note on the go (Shortcut on iPhone → saves to iCloud Drive). Agent transcribes via Whisper API, extracts 2–3 content ideas with hooks, and appends them to `docs/Marketing/Idea Inbox.md` automatically. Never lose a raw insight again.
- **Tool / MCP:** Apple Shortcuts + OpenAI Whisper API + Claude for idea extraction + filesystem write
- **Time saved:** ~2 hours/week (capturing 10x more fleeting ideas, reducing blank-page content planning)
- **Difficulty:** Easy — Shortcuts automation + Whisper is well-documented

### 3. App Store Keyword & Competitor Monitor
- **Area:** Product / ASO
- **What to automate:** Weekly crawl of top 10 competitors' App Store listings (keywords, ratings, review themes, price changes). Agent diffs week-over-week changes, identifies new keyword opportunities, and outputs a `ASO-weekly-[DATE].md` report to `output/Research/`. Catches competitor moves before they affect your ranking.
- **Tool / MCP:** AppFollow API or Sensor Tower export + Claude analysis + filesystem
- **Time saved:** ~3 hours/week of manual monitoring
- **Difficulty:** Medium — depends on ASO data API access (AppFollow has a free tier)

### 4. Post-Launch Review Response Agent
- **Area:** Customer Feedback / Product
- **What to automate:** After Bloggo launches, agent polls App Store Connect API daily for new reviews. 1–2 star reviews get a warm, human draft response for Yoobin to approve in 30 seconds. 4–5 star reviews: extracts the praised feature and appends it to `output/Content/User Love.md` for use in future marketing copy.
- **Tool / MCP:** App Store Connect API + Claude + Notion MCP (to log to Content Engine)
- **Time saved:** ~2 hours/week post-launch
- **Difficulty:** Medium — App Store Connect API requires App Store Connect auth token

### 5. Weekly Finance & Metrics Snapshot Agent
- **Area:** Finance / Business Health
- **What to automate:** Every Monday morning, agent pulls: App Store Connect analytics (downloads, sessions, crashes), simulated paper portfolio P&L (from this briefing's stock sim), and social engagement metrics (manually pasted or via API). Outputs a one-page `finance-snapshot-[DATE].md` to `Magic/`. Replaces ad-hoc dashboard checking.
- **Tool / MCP:** App Store Connect API + Claude + filesystem (+ Stripe API post-monetization)
- **Time saved:** ~1.5 hours/week (replaces fragmented dashboard checking)
- **Difficulty:** Easy to start (manual paste) → Medium (with full API integration)
